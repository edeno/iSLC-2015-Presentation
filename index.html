HOW: What is D3?

D3 is a low level javascript based library for turning data into web elements


Data is stored as a property of a web element.

If dataset changes, D3 can handle those changes easily by allowing you to decide what to do with:

  • new data not bound to a web element (data enter)
  • existing data bound to a web element (data join)
  • data that no longer exists bound to a element (data exits)

Once data is bound to a web element, we can change properties of the web element based on the data

D3 Pros and Cons

    Pros
  • Web based - easy to share
  • Flexible styling - web design constantly evolving
  • Easy to save static figures - SVG
    Cons
  • Learning Curve
  • Takes more time
  • Additional complexity
